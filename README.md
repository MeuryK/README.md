# Projeto: Cadastro de Usuários com Python e SQLite

## Objetivo
Sistema simples para cadastrar, listar, atualizar e deletar usuários usando Python e SQLite.

## Funcionalidades
- Criar usuários
- Listar usuários
- Atualizar dados
- Deletar registros

## Banco de Dados
Tabela: `usuarios`
- id
- nome
- email
